# دستفروش — قالب HTML معرفی فروشنده‌های سیار

این پروژه یک قالب استاتیک HTML برای معرفی فروشنده‌های سیار است. تمرکز پروژه روی نمایش پروفایل‌ها، دسته‌بندی‌ها، لوکیشن و راه ارتباطی است و هیچ پرداخت یا خرید آنلاینی داخل سایت انجام نمی‌شود.

## ویژگی‌ها
- طراحی RTL و فارسی
- واکنش‌گرا برای موبایل و دسکتاپ
- استفاده از Tailwind CDN و فونت `Vazirmatn`
- فرم‌های نمونه آماده اتصال به بک‌اند

## صفحات
- `index.html` صفحه اصلی و معرفی پلتفرم
- `auth.html` ورود / ثبت‌نام فروشنده‌ها
- `categories.html` نمایش دسته‌بندی‌ها
- `vendor-public.html` پروفایل عمومی فروشنده
- `vendor-location.html` پروفایل + لوکیشن
- `vendor.html` پنل فروشنده (پروفایل، لوکیشن، نظرات)
- `vendors-map.html` نمایش فروشنده‌ها روی نقشه
- `account-edit.html` ویرایش حساب کاربری

## اجرا
این پروژه نیاز به بیلد ندارد. کافی است فایل‌های HTML را در مرورگر باز کنید. نقطه شروع پیشنهادی:
- `index.html`

اگر می‌خواهید همه چیز را مثل یک سایت استاتیک بالا بیاورید، می‌توانید یک سرور ساده محلی اجرا کنید و سپس `index.html` را باز کنید.

## نکات
- فرم‌ها نمونه هستند و برای اتصال به API یا بک‌اند باید منطق ارسال داده را اضافه کنید.
- تمام استایل‌ها داخل خود فایل‌ها و با Tailwind CDN تعریف شده‌اند.
